# Features Overview

> Complete guide to AAFairShare features

## Core Features

### Expense Tracking

Track daily expenses with full details including amount, date, category, location, and optional receipt attachments.

**Key Capabilities:**

- Add/edit/delete expenses
- Categorize by type (Dining, Transport, etc.)
- Tag with locations
- Attach receipts
- Split 50/50 or custom

[Learn more →](./expenses.md)

### Recurring Expenses

Automate tracking for regular bills and subscriptions.

**Key Capabilities:**

- Set frequency (weekly, monthly, yearly)
- Optional end date
- Auto-generate expenses
- Track upcoming payments

[Learn more →](./recurring.md)

### Settlement Tracking

Keep track of who owes whom and settle up easily.

**Key Capabilities:**

- Real-time balance calculation
- One-click settlement
- Settlement history
- Undo settlements

[Learn more →](./settlements.md)

### Savings Goals

Track shared savings towards common goals.

**Key Capabilities:**

- Create joint goals
- Track contributions by user
- Milestone celebrations
- Target date tracking
- Progress visualization

[Learn more →](./savings-goals.md)

### Analytics

Understand spending patterns with visual analytics.

**Key Capabilities:**

- Category breakdown
- Location analysis
- Monthly trends
- User comparison
- Year-end summary

[Learn more →](./analytics.md)

### Receipt Management

Organize and store receipts digitally.

**Key Capabilities:**

- Attach to expenses
- Standalone uploads
- Search and filter
- Secure storage

[Learn more →](./receipts.md)
