# First Steps

> Learn to use AAFairShare effectively

## Logging In

1. Navigate to the app URL
2. Enter your email and password
3. Click "Sign In"

> **Note:** Registration is closed. Contact the admin to create your account.

## Dashboard Overview

After logging in, you'll see the Dashboard with:

- **Monthly Summary** - Total expenses, who paid what, settlement status
- **Recent Expenses** - List of expenses for the current month
- **Quick Actions** - Add expense button (floating action button on mobile)

## Adding Your First Expense

### Method 1: Quick Add (Mobile)

1. Tap the `+` floating action button
2. Fill in the expense details
3. Tap "Save Expense"

### Method 2: Navigation (Desktop)

1. Click "Add Expense" in the sidebar
2. Complete the form
3. Click "Save Expense"

### Expense Fields

| Field       | Required | Description                               |
| ----------- | -------- | ----------------------------------------- |
| Amount      | Yes      | Expense amount in £                       |
| Date        | Yes      | When the expense occurred                 |
| Category    | Yes      | Type of expense (Dining, Transport, etc.) |
| Location    | No       | Where the expense occurred                |
| Split Type  | Yes      | How to split (50/50 or 100% mine)         |
| Description | No       | Additional notes                          |
| Receipt     | No       | Upload a receipt image                    |

## Understanding Split Types

- **50/50** - Both partners share equally
- **100% Mine** - Only you are responsible (e.g., personal item)

## Viewing the Balance

The Dashboard shows:

- **You Paid**: Total amount you've paid
- **Your Share**: What you should have paid
- **Settlement**: Who owes whom and how much

## Settling Up

1. Go to **Settlement** page
2. View the calculated balance
3. When you pay/receive money, click "Mark as Settled"
4. The settlement is recorded in history

## Key Features to Explore

### Recurring Expenses

Set up automatic expense tracking for:

- Monthly subscriptions
- Regular bills
- Rent payments

Navigate to **Recurring** → Click "Add Recurring"

### Savings Goals

Track shared savings:

1. Go to **Savings Goals**
2. Click "New Goal"
3. Add contributions as you save

### Analytics

View spending patterns:

- Category breakdown
- Location breakdown
- Monthly trends
- Year-end summary

### Receipts

Organize your receipts:

- Attach to expenses
- Upload standalone receipts
- Search and filter

## Navigation

### Desktop

Use the sidebar navigation or keyboard shortcuts.

### Mobile

Use the bottom navigation bar:

- Home (Dashboard)
- Settlement
- Savings
- Analytics
- Settings

## Tips for Success

1. **Log expenses immediately** - Don't let them pile up
2. **Use categories consistently** - Better analytics
3. **Add locations** - Track where you spend most
4. **Settle monthly** - Keep balances manageable
5. **Check analytics** - Identify spending patterns
